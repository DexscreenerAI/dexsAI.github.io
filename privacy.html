Privacy Policy

Last updated: March 28, 2026

✓ Privacy First

TL;DR: DexScreener AI does NOT collect, store, or transmit any personal data. Everything runs locally in your browser.

Overview

DexScreener AI is a Chrome extension that provides AI-powered token analysis directly on DexScreener.com. We are committed to protecting your privacy and being transparent about our practices.

Data Collection

We do NOT collect:

  • Personal information (name, email, etc.)
  • Wallet addresses
  • Browsing history
  • Trading activity
  • Any other user data

How the Extension Works

DexScreener AI operates entirely within your browser:

  • Token data is fetched directly from the public DexScreener API
  • Rugcheck data (for Solana tokens) is fetched from the public Rugcheck API
  • Analysis is performed locally in your browser using JavaScript
  • No data is sent to our servers (we don't have any servers)

Third-Party Services

The extension makes API calls to:

  • DexScreener API (api.dexscreener.com) — to fetch token/pair data
  • Rugcheck API (api.rugcheck.xyz) — to fetch Solana token risk data

These are public APIs. We do not control their privacy policies. Please review their respective privacy policies for more information.

Local Storage

The extension may store minimal preferences locally in your browser (such as panel position or collapsed state). This data never leaves your device.

Permissions

The extension requests the following permissions:

  • Host permissions for dexscreener.com — to inject the analysis panel
  • Host permissions for API endpoints — to fetch token data

We request only the minimum permissions necessary for the extension to function.

Children's Privacy

DexScreener AI is not intended for use by children under the age of 18. We do not knowingly collect any information from children.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated revision date.
