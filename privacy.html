Privacy Policy

Last updated: April 2, 2026

✓ Privacy First

TL;DR: $ByeBoss does NOT collect, store, or transmit any personal data. Everything runs locally in your browser.

Overview

$ByeBoss is a crypto token intelligence platform available as a web application (dexscreenerai.com) and a Chrome extension. The platform includes tools such as Token Analysis, AI Buy/Sell/Target analysis, Token Comparator, Gem Hunter, Sniper Bot, and OG Detector. We are committed to protecting your privacy and being transparent about our practices.

Data Collection

We do NOT collect:

  • Personal information (name, email, etc.)
  • Wallet addresses or private keys
  • Browsing history
  • Trading activity or portfolio data
  • IP addresses or device fingerprints
  • Any other user data

We do not use cookies, tracking pixels, analytics services, or any form of user tracking.

How the Platform Works

Web Application (dexscreenerai.com)

The web application runs entirely in your browser:

  • Token data is fetched from the public DexScreener and CoinGecko APIs
  • Safety checks are performed via the public Rugcheck API
  • AI analysis (Buy/Sell, Target, Strategy) is processed through our proxy backend — only the token symbol and market data are sent, never any personal information
  • Gem Hunter scans trending tokens using public APIs and scores them locally
  • Token Comparator compares tokens using public market data and safety APIs
  • Sniper Bot monitors new token listings using public APIs — all settings are stored locally in your browser
  • OG Detector analyzes early wallet activity using public blockchain data
  • AI response cache is stored temporarily in your browser memory (5 minutes) and is never sent externally

Chrome Extension

The Chrome extension operates entirely within your browser:

  • Token data is fetched directly from the public DexScreener API
  • Rugcheck data (for Solana tokens) is fetched from the public Rugcheck API
  • Analysis is performed locally in your browser using JavaScript
  • Panel preferences (position, collapsed state) are stored locally and never leave your device

Third-Party Services

The platform makes API calls to the following public services:

  • DexScreener API (api.dexscreener.com) — token and pair market data
  • CoinGecko API (api.coingecko.com) — token market data for major cryptocurrencies
  • Rugcheck API (api.rugcheck.xyz) — token safety and risk analysis
  • AI Backend Proxy — processes AI analysis requests (only receives token symbol and public market data, no personal information)

These are third-party services with their own privacy policies. We do not control how they handle requests. Only publicly available market data is transmitted — never any personal or identifying information.

Local Storage

The platform stores minimal data locally in your browser:

  • Sniper settings — chain preferences, risk parameters, and scanner configuration
  • Extension preferences — panel position and display state
  • AI cache — temporary in-memory cache of AI responses (cleared after 5 minutes or page refresh)

This data is stored exclusively on your device and is never transmitted externally.

Permissions (Chrome Extension)

The extension requests the following permissions:

  • Host permissions for dexscreener.com — to inject the analysis panel on token pages
  • Host permissions for API endpoints — to fetch token and safety data

We request only the minimum permissions necessary for the extension to function.

No Account Required

$ByeBoss does not require any account creation, login, or registration. There are no user accounts, passwords, or authentication of any kind.

Children's Privacy

$ByeBoss is not intended for use by children under the age of 18. We do not knowingly collect any information from children.
